Test MyList class
=================

Import the class:
    >>> MyList = __import__('1-my_list').MyList

Test normal list:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test empty list:
    >>> my_list2 = MyList()
    >>> my_list2.print_sorted()
    []

Test already sorted:
    >>> my_list3 = MyList()
    >>> my_list3.append(1)
    >>> my_list3.append(2)
    >>> my_list3.append(3)
    >>> my_list3.print_sorted()
    [1, 2, 3]

Test negative numbers:
    >>> my_list4 = MyList()
    >>> my_list4.append(-1)
    >>> my_list4.append(-5)
    >>> my_list4.append(-3)
    >>> my_list4.print_sorted()
    [-5, -3, -1]

Test single element:
    >>> my_list5 = MyList()
    >>> my_list5.append(42)
    >>> my_list5.print_sorted()
    [42]
